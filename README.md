# Dharma Ninja E-commerce Platform 🌐

## 🌐 Live Website
https://dharmaninjaimports.com/

## 📌 Description
This is a full-stack eCommerce platform developed for an international client. The platform allows users to browse products, place orders, and make donations.

## 💼 Real-World Application
- Developed for an international client
- Used for product sales and donation management

## 🚀 Features
- Product listing and ordering system
- Donation page integration
- Integrated Razorpay/Stripe payment gateway for secure transactions
- Implemented Laravel authentication system with secure password handling
- Admin dashboard for managing products, offers, and content
- Real-time analytics dashboard displaying product revenue and inventory (stock) management
- Responsive UI

## 🛠️ Tech Stack
- Laravel
- MySQL
- HTML, CSS, JavaScript
